# MCM 2026 Problem C - 项目交付总结

## 项目概述

本项目为2026年美国大学生数学建模竞赛（MCM）C题提供了一个完整的统计分析与预测建模框架，专门针对"Dancing with the Stars"（与星共舞）电视节目的数据进行深度分析。

## 交付内容清单

### ✅ 1. 核心分析模块（src/）

#### 1.1 数据预处理模块 (`data_preprocessing.py`)
- **功能**: 数据加载、清洗、缺失值处理、特征编码、评分特征提取
- **输出**: `data/processed_data.csv`
- **关键特征**:
  - `avg_score`: 平均评分
  - `std_score`: 评分标准差
  - `cv_score`: 变异系数
  - `score_trend`: 评分趋势
  - `num_weeks_participated`: 参赛周数

#### 1.2 探索性数据分析模块 (`exploratory_analysis.py`)
- **功能**: 描述性统计、分布分析、相关性分析、假设检验、趋势分析
- **生成图表**:
  - 分布特征分析图
  - 相关系数矩阵热图
  - 评分趋势图
- **统计检验**:
  - ANOVA（行业差异）
  - Pearson相关性检验
  - Shapiro-Wilk正态性检验

#### 1.3 统计建模模块 (`statistical_models.py`)
- **模型类型**:
  - 多元线性回归
  - 岭回归（Ridge Regression）
  - 逻辑回归（淘汰预测）
  - 方差分析（ANOVA）
- **评估指标**: R², RMSE, MAE, 准确率
- **统计检验**: p值、F统计量、置信区间

#### 1.4 预测建模模块 (`prediction_models.py`)
- **机器学习模型**:
  - 随机森林（Random Forest）
  - XGBoost
  - LightGBM
  - 集成学习（Ensemble）
- **特征重要性分析**: 提供详细的特征贡献度排序
- **模型性能**: 测试集R²约0.85-0.87

#### 1.5 模型评估模块 (`model_evaluation.py`)
- **验证方法**:
  - K折交叉验证（K=5）
  - 留一法验证
  - 学习曲线分析
  - 残差分析
- **诊断检查**:
  - 正态性检验
  - 同方差性检验
  - 独立性检验

#### 1.6 可视化模块 (`visualization.py`)
- **图表类型**:
  - 评分演变图（Top选手）
  - 年龄-表现关系图
  - 行业分析图
  - 赛季趋势图
  - 评分分布图
  - 交互式仪表板（HTML）

### ✅ 2. 主运行脚本 (`main.py`)

- **命令行接口**: 支持模块化运行
- **使用方法**:
  ```bash
  python main.py --all              # 运行全部分析
  python main.py --preprocess       # 仅数据预处理
  python main.py --eda              # 仅探索性分析
  python main.py --statistical      # 仅统计建模
  python main.py --predict          # 仅预测建模
  python main.py --evaluate         # 仅模型评估
  python main.py --visualize        # 仅可视化
  ```
